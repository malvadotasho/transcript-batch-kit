# Contributing

Thanks for considering a contribution.

## Local Setup

```powershell
python -m pip install -e .
python -m pytest
```

## Pull Requests

Please keep changes small and focused. A useful pull request usually includes:

- a short description of the problem
- the implementation change
- tests or a clear manual verification note

## Good First Issues

Ideas that would fit the project:

- add JSON manifest output
- support custom media extensions
- add optional `.docx` export when `python-docx` is installed
- add duplicate transcript detection
